# Data-Science-Assignment
Data Science assignment including classification, clustering, and forecasting tasks
# Task 1: Bank Marketing Prediction

## Objective
The objective of this task is to predict whether a customer will subscribe to a term deposit based on demographic and marketing campaign data.

## Dataset
Bank Marketing Dataset (Kaggle)

Target Variable:
- deposit (Yes / No)

## Methodology
1. Loaded the dataset from Kaggle.
2. Cleaned and preprocessed the data.
3. Encoded categorical variables.
4. Split data into training and testing sets.
5. Trained a classification model.
6. Evaluated performance using standard metrics.

## Model Used
- Random Forest Classifier

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score

## Conclusion
The model successfully predicts customer subscription behavior and can assist banks in improving targeted marketing strategies.
